# cat

- `cat-poses.svg` — чистый SVG-лист с 5 позами котёнка.
- `index.html` — страница предпросмотра с **inline SVG** (для случаев, когда превью открывает только `/`).

Локальный запуск:

```bash
python3 -m http.server 8000
```

Открыть в браузере:
- `http://localhost:8000/` (рекомендуется для preview)
- `http://localhost:8000/cat-poses.svg`
